Гирлянда на адресных светодиодах.
Идея: накрутить гирлянду как попало, разбить светодиоды на блоки и зажигать их по программе.

Программа - набор троек:
* Сетка
* Эффект
* Количество повторов

Сетка - разбиение светодиодов на группы. Сетки могут быть 1..3-мерными. Внутри группы светодиоды упорядочены для того, чтобы группы были одинаковы. 
Например, вертикальные или горизонтальные полосы.

Эффект - последовательность включения групп. Определяет допустимую размерность сетки.
Возможные эффекты:
* Волна - зажигаем группы по очереди ...-0%-0%-50%-100%-50%-0%-0%-...., сдвигаем.
* Флаг - зажигаем по очереди группы 0..n-1 в белый, 0..n-2 в синий, 0..n-3 в красный. Потом, с n-1 - по 2 каждого цвета, по 3 и т.д.

Архитектура:
* контроллер управляет ирляндой. Параметры получает от сервера параметров.
* сервер параметров хранит и редактирует параметры.

Контроллер - esp32
Сервер - raspberry py, python, flask, SQLite.
АРМ - vue.js

Режимы работы контроллера:
* Создание сетки:
* Выполнение программы

Создание сетки:
* Определям размерность и размеры, в результате получаем набор групп.
* По очереди зажигаем светодиоды, для каждого определяем группу.
* Сохраняем с определенным именем.
